Front-end Technologies

What is Frontend Development?

Front end development is the process of building the user interface of an application. It includes everything that the user sees and interacts with—text, images, video, and audio—basically the visual appearance.

What are Frontend Technologies?

Backend and front end technologies are the two main categories that make up a web application. A web application is a piece of software that runs on a server and provides functionality to users, such as sending email or creating online forms. While front end technologies deal with the appearance, a backend technology is what powers the user interface—how it’s built and maintained.
